ScireLibros

Site que recebeu o nome de ScireLibros, trata-se de uma loja de livros fictícia. Desenvolvido seguindo o conceito do Mobile First, possui breakpoint no seu visual para Desktop e Mobile, seus principai elementos por seguir esta ideia ocorrem na responsividade dos elementos, seja na mudança de tamanho ou em sua distribuição no site, também possui uma funcionalidade implementada por meio de Javascript para fazer um menu responsivo para Mobile. Além de também possuir um menu dinâmico.

Como dito, ele foi desenvolvido usando Javascript, além de outras tecnologias, como HTML, CSS, e hospedado usando o Github Pages.
